# 🌍 Desi Explorer

**Desi Explorer** is a visually rich travel web application that helps users explore popular Indian cities and discover destinations based on their mood and interests.

---

## 🚀 Features

* 🏙️ Multiple city pages (Delhi, Mumbai, Amritsar, Goa, Manali, Varanasi, Jaipur, Jabalpur)
* 🎨 Modern UI with glassmorphism design
* 🔍 Search functionality for quick city discovery
* 🎯 Category-based filters (Historical, Spiritual, Nature, Fun)
* 🤖 Mood-based place recommendation system
* 📱 Responsive layout
* ✨ Smooth UI interactions and animations

---

## 🧠 Smart Recommendation System

Users can select a mood and the app will:

* Suggest a relevant place
* Scroll to that section
* Highlight the recommended location

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Reusable components, Glassmorphism UI)
* **JavaScript** – Interactivity (Search, Filters, AI logic)
